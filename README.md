# SaferClaw Transfer 🦞

**Free one-time tool to clone your OpenClaw setup from one Mac to another.**

## What it does

- **Backup** — packages `~/.openclaw/`, `~/Projects/`, npm-global openclaw module, and key dotfiles into a single `.tar.gz` on your Desktop
- **Restore** — extracts the backup on your new Mac, installs Homebrew/Node/OpenClaw if missing, starts the gateway

## Usage

1. Download [SaferClaw-Transfer.dmg](https://clawrenceisthebestboy-dev.github.io/saferclaw-transfer/SaferClaw-Transfer.dmg)
2. Open it, drag the app to Applications
3. On old Mac: click **Backup This Mac**
4. AirDrop the `.tar.gz` to new Mac
5. On new Mac: click **Restore From Backup**

## Requirements

- macOS 14+
- Apple Silicon (arm64)

Before You Install

SaferClaw Transfer — Setup Guide

Hey there! 👋 Before you open the app, let's make sure your Mac is ready to go. This takes about 2 minutes and will save you a lot of headaches later. Promise!

1

🔑 Make sure you're an Administrator

You need admin rights to install things on your Mac.

  1. Click the Apple menu (🍎) in the top-left corner
  2. Go to System Settings → Users & Groups
  3. Find your name — it should say "Admin" underneath it
💡 If it doesn't say Admin, ask whoever set up the Mac to help — or contact us!
2

🔓 Allow apps from identified developers

macOS can be a little overprotective. Let's loosen that up just a tiny bit.

  1. Click Apple menu → System Settings
  2. Go to Privacy & Security
  3. Scroll down to "Allow apps downloaded from:"
  4. Select "App Store and identified developers"
✅ This is totally safe — it's the standard setting most Mac users have.
3

🛡️ When macOS says it can't verify the app

The first time you open SaferClaw Transfer, macOS might show a warning saying "Apple cannot verify this app." Don't panic!

  1. Click "Open Anyway" (not Cancel!)
  2. If you don't see it — go back to System Settings → Privacy & Security and scroll down. The button will appear there.
💡 This happens because we're an indie app. The app is completely safe — it's just small-developer stuff!
4

🤖 Say yes to Automation permissions

When SaferClaw Transfer asks to control other apps (like Terminal), macOS will ask permission.

  1. A popup will appear: "SaferClaw Transfer wants to control Terminal"
  2. Click Allow
✅ This is how the app runs the installer script for you automatically — without it you'd have to do everything by hand.
5

🚀 Run SaferClaw-Installer.command FIRST

This is the most important step! The installer sets up everything your Mac needs before you restore.

  1. Double-click the SaferClaw-Installer.command file
  2. A Terminal window will open
  3. Type your Mac password when asked (the cursor won't move — that's normal!)
  4. Wait for it to say "DONE!" before moving on
⏱️ This usually takes 2–5 minutes depending on your internet speed. Grab a coffee! ☕
6

✅ NOW open SaferClaw Transfer and click Restore

Once the installer is done:

  1. Open the SaferClaw Transfer app
  2. Click "I'm setting up a NEW Mac"
  3. Follow the checklist, then click "I'm Ready — Select Backup File"
  4. Pick your backup .tar.gz file (AirDropped from your old Mac)
  5. Sit back and watch the magic happen! 🪄

Ready to go? Head back to the main page to download SaferClaw Transfer.
